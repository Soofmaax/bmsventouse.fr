# Quick Start

Get the project running in three steps:

1) Clone
```bash
git clone https://github.com/Soofmaax/bmsventouse.fr.git
cd bmsventouse.fr
```

2) Serve (choose one)
```bash
# Node
npx http-server -p 8080

# or Python
python3 -m http.server 8080
```

3) Open
- http://localhost:8080

That’s it — no build step required.
