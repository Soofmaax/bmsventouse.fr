# Installation

This is a static website. No build step is required to view the site locally.

## Prerequisites

- Node.js 20+ (or Python 3)
- Git (to clone the repository)

## Steps

1) Clone the repository
```bash
git clone https://github.com/Soofmaax/bmsventouse.fr.git
cd bmsventouse.fr
```

2) Option A — Start a local server with Node
```bash
npx http-server -p 8080
```

2) Option B — Start a local server with Python
```bash
python3 -m http.server 8080
```

3) Visit
- http://localhost:8080
