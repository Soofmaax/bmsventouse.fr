Comment se passe votre demande de devis ?

Simple, rapide, sans engagement : on vous accompagne du premier brief jusqu’au lancement du dispositif.

1. Vous nous briefez

  • Vous décrivez votre projet en 4–5 points (lieux, dates, horaires, type de projet).
  • Vous précisez vos besoins : ventousage, sécurité, gardiennage, convoyage…
  • Vous indiquez le niveau d’urgence et les contraintes sensibles.

2. On vous appelle si besoin

  • On vérifie avec vous les détails clés pour éviter les surprises sur le terrain.
  • On vous propose les options utiles (sécurité, gardiennage, convoyage, loges…).
  • Si vous le souhaitez, on peut organiser un repérage ou un échange vidéo.

3. Vous recevez un devis clair

  • Devis détaillé poste par poste (ventousage, signalisation, sécurité, etc.).
  • Tarifs directs, sans surcoût d’agence, avec les options identifiables.
  • Délais de mise en place et périmètre bien expliqués, prêts à être validés.

Les infos utiles pour un devis précis

Pour aller vite et éviter les allers‑retours, pensez à préciser :
  • la ville et les rues concernées,
  • les dates, horaires et la durée (une journée, plusieurs jours ou plusieurs semaines),
  • le type de projet (film, série, publicité, shooting, défilé, événement),
  • les besoins principaux (ventousage, sécurité plateau, gardiennage, convoyage de véhicules ou décors),
  • et, le cas échéant, si le projet alterne entre la France et la Belgique.
Vous pouvez nous transmettre ces éléments par téléphone, WhatsApp ou via la page Contact. Nous répondons en général sous 24 h ouvrées.
